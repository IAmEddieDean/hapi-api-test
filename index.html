Path Parameters Options Description
/items N/A Sorting parameters as a query string, e.g.
/items?price=1		 Return a complete collection of all items in the db, optionally sorted via query string: prop=1/-1, where 1 = descending order, -1 = ascending order.
/items/{id} id: a MongoDB object id for an individual item N/A Return a single record, or code 204 if none found.
/items/{long}/{lat}/{dist?} long: longitudinal coordinate
lat: a latitudinal coordinate
dist: the maximum distance, in miles, to search for items, optional, defaults to 50. 		Sorting parameters as a query string, e.g.
/items?price=1		 Return a collection of items within a radius of the client, default sorted by distance, optionally sorted by other properties in a query string. If no items found, return code 204.
/users/{id}/items id: a MongoDB object id for a user Sorting parameters as a query string, e.g.
/items?price=1		 Return a collection of items from a single user, optionally sorted by other properties in a query string. If no items found, return code 204.
/reset N/A N/A Erase all records from the DB and reset based on static data provided.
